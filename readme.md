### hzview

A simple audio visualizer that runs in the terminal.

### Usage

Compile it using the makefile and run it.

+ to increase sensitivity
- to decrease sensitivity
q to quit

### About

Uses the AudioToolbox library and Accelerate. Quitting using <C-c> might make your cursor invisible so beware of that.
